---
description: Build and start Next.js production server for MatriSense frontend
---

1. Open a terminal in the frontend directory.
   // turbo
   ```
   cd d:\\MatriSense\\frontend
   ```
2. Install dependencies (if not already installed).
   // turbo
   ```
   npm install
   ```
3. Build the Next.js application.
   // turbo
   ```
   npm run build
   ```
4. Start the production server.
   // turbo
   ```
   npm start
   ```
