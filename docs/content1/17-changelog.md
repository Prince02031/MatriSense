# Changelog

All notable changes to the MatriSense system are tracked below.

### Version 1.0.0 (June 2026)
*   **Initial MVP Launch:** Complete triage queue and verification framework.
*   **Patient Triage Workflow:** Real-time symptom submission in natural Bangla, AI extraction, and clinical rule classification.
*   **Health Worker Dashboard:** Core queue, case detail overlays, map lookup, and status update cards.
*   **Regional Referral System:** Seeding of 15+ Bangladesh clinics, distance-sorted assignment, and patient referral notifications.
*   **Profile Management:** Gestational age logging, document upload, and worker verification requests.
*   **Safety Guardrails:** Dynamic post-generation check blocking specific diagnostics/drugs and enforcing Consult Doctor disclaimers.

### Version 0.9.0 (May 2026)
*   **Rule-Aware RAG Integration:** Retrieve matched guidance based on active clinical rules and risk level.
*   **LLM Explanation Display:** Render natural language explanations on why symptoms were extracted and what each warning sign means.
*   **Health Worker Verification Request:** Initial UI forms for professional registration.
*   **Patient GPS Capture:** Optional automatic coordinate tracking on profile creation.

### Version 0.8.0 (April 2026)
*   **Basic Triage Engine:** Simple conditional logic maps symptoms to categories.
*   **Patient History:** Simple list of previous entries for a logged-in patient.
*   **Health Worker Case List:** Static table displaying mock patient entries.
*   **Bangla Symptom Input:** First-generation input forms supporting manual symptom checklists.
